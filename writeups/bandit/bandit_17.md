## Bandit Level 17

## Level goal
The credentials for the next level can be retrieved by submitting the password of the current level to **a port on localhost in the range 31000 to 32000**. First find out which of these ports have a server listening on them. Then find out which of those speak SSL/TLS and which donâ€™t. There is only 1 server that will give the next credentials, the others will simply send back to you whatever you send to it.

## Solution
    - used **nmap localhost -p 31000-32000** to find the ports that have a server listening on them (are open) in the given range
    - used **ncat --ssl localhost "portnumber"** followed by the password from the previous level on each resulting port until we found one that returned a RSA private key

## New concepts learned
    - nmap = a tool for network discovery and security auditing
        Scans networks to find active devices
        Discovers open ports and services
        Identifies operating systems
        Detects vulnerabilities

    # Basic port scan
        nmap 192.168.1.1
    # Scan specific ports
        nmap -p 22,80,443 target.com
    # OS detection
        nmap -O target.com
    # Aggressive scan (includes scripts, version detection)
        nmap -A target.com

    
    - ncat = Networking utility for reading/writing network connections
        Creates TCP/UDP connections
        Transfers files over networks
        Port forwarding and proxying
        Network debugging and testing

    # Connect to a server
        ncat example.com 80
    # Listen for incoming connections (server mode)
        ncat -l 8080
    # File transfer (receiver)
        ncat -l 8080 > received_file.txt
    # File transfer (sender)
        ncat receiver.com 8080 < file_to_send.txt
